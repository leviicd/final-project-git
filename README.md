# Emotion Detector

## Project Description
This project is a web-based application developed using **Python** and **Flask**. It utilizes the **Watson NLP library** to perform emotion detection on text provided by the user. The application analyzes the input and returns scores for five emotions: anger, disgust, fear, joy, and sadness, identifying the dominant emotion among them.

## Features
- Text-based emotion analysis.
- Web interface built with Flask.
- Integration with IBM Watson NLP API.
- Error handling for blank or invalid inputs.
- Static code analysis compliant (PyLint).

## How to Run
1. Clone the repository.
2. Install dependencies: `pip install requests flask`.
3. Run the server: `python3 server.py`.
4. Access via browser at `http://localhost:5000`.
